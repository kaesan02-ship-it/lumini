# ⚙️ Lumini 프로젝트 자동화 및 보안 규칙

이 프로젝트에서 Antigravity 에이전트는 사용자의 번거로움을 줄이기 위해 다음과 같은 '스마트 자동화' 원칙을 따릅니다.

## 1. ✅ 자동 실행 허용 범위 (SafeToAutoRun: true)
다음과 같은 일상적이고 안전한 작업은 사용자의 개별 승인 없이 자동으로 실행합니다.
- **코드 수정 및 리팩토링**: 기능에 큰 영향을 주지 않는 간단한 로직 수정, 스타일 변경, 파일 이동 등
- **패키지 설치 및 빌드**: 일반적인 라이브러리 추가(`npm install`), 빌드 확인(`npm run build`), 테마 적용 등
- **문서화**: README, 일지, 가이드북 업데이트 등
- **상태 확인**: `git status`, `ls`, `cat` 등 시스템 상태 조회 명령어

## 2. ⚠️ 명시적 동의 필수 범위 (SafeToAutoRun: false)
다음과 같은 중대하거나 파괴적인 작업은 반드시 사용자의 **'Accept'** 버튼 클릭을 통해 동의를 구합니다.
- **파일 삭제**: 프로젝트 소스 파일이나 중요 데이터의 영구 삭제
- **보안 및 민감 정보**: API 키 발급, 개인 정보 연동, 외부 계정 접근 등
- **시스템 설정 변경**: OS 환경 변수 수정, 시스템 전역 소프트웨어 설치(예: Git 설치 등)
- **대규모 파괴적 수정**: 프로젝트 아키텍처를 전면 뒤흔드는 파괴적인 변경 사항

## 3. 🎯 에이전트 다짐
사용자님이 일일이 버튼을 누르는 수고를 덜어드리되, 프로젝트의 안전은 최우선으로 지키겠습니다. 이제 더욱 빠르고 쾌적하게 작업하겠습니다! 🚀
