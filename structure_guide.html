Interactive Dashboard

루미니 인터랙티브 앱 맵 & 구조 가이드

로컬 개발 서버에서 한눈에 각 해시 페이지로 연결하여 흐름을 검증하고, 백엔드/프론트엔드 담당 소스 매핑을 확인할 수 있는 개발용 맵 대시보드입니다.

🧱 레고 블록식 단방향 아키텍처 (격리 설계)

루미니 웹 앱은 스파게티 코드를 완벽히 방지하도록 각 기능 단위가 엄격히 격리되어 있습니다. 화면을 띄워주는 Page, 작은 화면 단위의 Component, 복잡한 비즈니스 로직 연산을 대행해 주는 Hook, 그리고 데이터의 근원인 Store 및 Supabase가 각각 독립적으로 맞물려 결합합니다.

📱
1. Pages
화면의 골격을 담당하는 최상위 페이지 (src/pages)
🧩
2. Components
화면을 조립하는 단위 UI 컴포넌트 (src/components)
⚙️
3. Hooks
화면과 독립되어 동작하는 연산 로직 (src/hooks)
🗄️
4. Store / DB
Supabase CRUD 및 전역 상태 저장소 (src/store & src/supabase)
