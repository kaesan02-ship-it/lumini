# 🕰️ Lumini 프로젝트 Git 기록 및 보관 가이드 (v7.0 완결)

축하합니다! 이제 사용자님의 컴퓨터에 **Git**이 성공적으로 설치되었고, Lumini v7.0 완성본이 첫 번째 기록으로 안전하게 보관되었습니다.

## 1. 🎞️ 현재 저장된 기록 확인하기
터미널(PowerShell)에서 아래 명령어를 입력하면 우리가 함께 만든 역사를 볼 수 있습니다.
- 명령어: `git log --oneline`
- 결과 예시: `ebef171 feat: Lumini v7.0 Production Ready - Architecture Refactoring & Persistence Complete`

## 2. 🚀 앞으로 새로운 기록을 남기려면?
코드를 수정하신 후, 그 시점을 기록하고 싶으실 때마다 아래 순서대로 입력하세요.
1. `git add .` (수정된 모든 파일을 선택)
2. `git commit -m "수정 내용 설명"` (설명과 함께 기록 저장)

## 🌐 GitHub에 연결하기 (추천)
이제 Git이 설치되었으므로, GitHub 온라인 저장소에 코드를 올리는 것도 매우 쉬워졌습니다.
1. [GitHub](https://github.com/)에서 새 저장소(New Repository)를 만듭니다.
2. 화면에 나오는 `git remote add origin [주소]` 명령어를 복사해서 터미널에 입력하세요.
3. `git push -u origin master`를 입력하면 온라인에 안전하게 복사본이 올라갑니다.
