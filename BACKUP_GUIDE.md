# 🕰️ Lumini 프로젝트 안전 보관 가이드

프로젝트의 소중한 코드를 보관하고 기록을 남기는 가장 좋은 방법인 **Git**이 현재 컴퓨터에 설치되어 있지 않습니다. 대신 아래의 방법들을 통해 프로젝트를 안전하게 지키고 발전시켜 보세요!

## 1. 📂 수동 백업 방법 (지금 바로 가능)
가장 간단한 방법은 프로젝트 폴더 전체를 복사해서 보관하는 것입니다.
1. `c:\Users\kaesa\안티그래비티\lumini-app` 폴더를 오른쪽 클릭합니다.
2. '복사' 후 다른 안전한 곳(외장 하드, 클라우드 등)에 '붙여넣기' 하세요.
3. 폴더 이름을 `lumini-app_v7_backup` 처럼 버전별로 저장해두면 나중에 되돌아가기 쉽습니다.

## 2. 🚀 Git 설치하여 '진짜 시간여행' 하기
Git을 설치하면 제가 처음에 말씀드린 '기록 관리' 시스템을 완벽하게 사용할 수 있습니다.
- **설치 방법**: [git-scm.com](https://git-scm.com/) 에 접속하여 Windows용 설치 파일을 다운로드하고 설치하세요.
- **설치 후**: 다시 저를 불러서 **"이제 Git 설치했어, 기록 남겨줘"**라고 하시면 제가 모든 내역을 정식으로 박제해 드릴게요!

## 3. 🌐 GitHub에 올리기 (추천)
[GitHub](https://github.com/) 계정을 만드시면 코드를 인터넷 공간에 무료로 저장할 수 있습니다. 
- 컴퓨터가 고장 나도 코드는 안전하며, 전 세계 누구에게나 내 프로젝트를 자랑할 수 있습니다.
