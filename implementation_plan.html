대시보드 성향 리포트 정상화 및 AI 에이전트 본부 완전 제거 계획

기획 및 변경 요지
💡 핵심 요약
대시보드 상에서 '내 성향 리포트' 카드가 회색 빈 화면으로만 보이고 우측 뱃지가 '분석 대기 중' 상태로 고정되는 오류를 파싱 불일치 및 CSS 차원(Recharts)에서 완전히 복구합니다. 더불어 사장님께서 불필요하다고 지적해주신 'AI 에이전트 본부' 기능의 흔적을 소스 코드와 라우트 상에서 물리적으로 완전 지워 경량화합니다.
상세 변경 파일 (Proposed Changes)
src/pages/DashboardPage.jsx Modify

`SoulReportCard`에 주입되는 `mbtiType` 바인딩 변수를 `profile?.mbti`에서 `profile?.mbti_type`으로 매칭해주어 뱃지 및 타이틀 파싱 오류를 잡습니다.

src/components/dashboard/SoulReportCard.jsx Modify

`RadarChart`를 감싸고 있는 flex 컨테이너에 `width: '100%'` 스타일을 강제 삽입하여, 가로 폭 미감지 현상에 의한 하얀 빈 공간 노출을 원천 해결합니다.

src/router/AppRouter.jsx Modify

`AgentHubPage`에 대한 import 선언, 허용된 딥링크 경로(`'agent-hub'`), 그리고 렌더링에 매핑되는 switch 조건문을 완벽히 삭제하여 의존성을 정리합니다.

src/pages/AgentHubPage.jsx Delete

더 이상 사용하지 않는 AI 에이전트 협업 본부 페이지의 컴포넌트 파일을 물리 디스크 상에서 안전하게 제거합니다.

검증 계획 (Verification Plan)
  • 자동화 빌드 검증: `npm run build`를 동작시켜 코드 수정 시 빌드가 깨지는 오류를 컴파일 레벨에서 원천 예방합니다.
  • 가상 브라우저 테스트: `node test_features.js` (Puppeteer)를 구동하여 렌더링 상의 crash 혹은 Javascript reference error 발생 여부를 체크하고 대시보드 돔 구조를 덤프해 대조합니다.
